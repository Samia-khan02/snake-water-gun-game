# Snake-Water-Gun Game

A Python-based Snake-Water-Gun game where the user plays against the computer.

## Features
- Random computer choice
- Score tracking
- Multiple rounds
- Replay option

## Technologies Used
- Python
- Random Module

## How to Run
1. Clone the repository
2. Run the Python file
3. Play the game
